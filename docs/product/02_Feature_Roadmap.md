# Feature Roadmap

## Phase 1 - MVP Foundation (Completed)
- User auth
- Task CRUD
- Priority levels
- Due dates
- Tags
- Search / filter
- Sorting
- Bulk actions
- MongoDB persistence

## Phase 2 - Reliability (Completed)
- Offline mode
- IndexedDB cache
- Sync queue
- Retry sync controls
- Pending sync states

## Phase 3 - Intelligence (Completed)
- Smart reminder timing
- Estimated duration planning
- Browser notification engine
- Reminder settings (buffer, toggles, quiet hours)
- Smart sections and workload insights

## Phase 4 - Gamification (Next)
- Growth tree
- Daily streaks
- Tree health states
- Rewards / unlockables

## Phase 5 - Advanced
- Analytics dashboard
- Collaboration
- Shared boards
- AI planning assistant
