# HaloTasks Context

HaloTasks is an offline-first productivity platform with smart scheduling, cross-device sync, reminders, and a gamified growth-tree system.

Core Pillars:
1. Fast Task Capture
2. Smart Planning
3. Reliable Sync
4. Habit Motivation
5. Beautiful UX

Current Implementation Snapshot:
- Full-stack auth and task CRUD
- Filters, sorting, and bulk productivity actions
- Smart sections (overdue, due today, upcoming, completed today, workload)
- Offline cache-first read + write queue + reconnect replay + retry
- Reminder engine with due-soon, overdue, and start-time intelligence
- Reminder settings panel (toggle, buffer, quiet hours, per-type controls)

Immediate Next Milestone:
- Growth Tree Phase 1 (logic + dashboard widget)
