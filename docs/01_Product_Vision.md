# HaloTasks v2 - Product Vision

## One-Line Summary
HaloTasks is a smart productivity platform that helps users complete tasks through scheduling intelligence, offline reliability, and habit-forming visual progress.

## Core Problem
Traditional todo apps store tasks but fail to help users finish them.

## Core Solution
HaloTasks plans backward from deadlines, syncs across devices, works offline, and rewards consistency through a living growth-tree system.

## Target Users
- Students
- Job seekers
- Professionals
- ADHD / forgetful users
- Productivity enthusiasts

## Key Differentiators
- Smart reminder engine
- Offline-first task management
- Cross-device sync
- Gamified growth tree
- Clean modern UX
